## 📌 About This Repository
This repository contains beginner-friendly notes and learning materials covering the fundamental concepts of Python programming.

It is designed as a quick reference guide for learners and developers who want to strengthen their Python basics.

---

## 🚀 Topics Covered

### 1. Python History
- Created by Guido van Rossum
- Released in 1991
- Designed for readability and simplicity

### 2. Python Features
- Easy to Learn
- Simple Syntax
- High-Level Language
- Interpreted Language
- Cross-Platform Support
- Large Standard Library
- Used in Web Development, AI, Data Science & Automation

3️⃣ Variables
Variables are used to store values.
Example
```python
name = "Python"
age = 25
price = 99.99
is_awesome = True
```
---
4️⃣ Identifiers
Identifiers are names given to variables, functions, classes, and objects.
Rules
Must start with a letter or underscore
Can contain letters, numbers, and underscores
Case-sensitive
Cannot be Python keywords
Examples
```python
user_name
totalMarks
_count
```
---
5️⃣ Data Types
Python supports multiple built-in data types.
Type	Description
int	Integer values
float	Decimal values
str	Text/String
bool	True/False
list	Ordered collection
tuple	Immutable collection
set	Unique values
dict	Key-value pairs
Example
```python
age = 22
price = 100.50
name = "Python"
```
---
6️⃣ Operators
Python supports various operators.
Arithmetic Operators
```python
+  -  *  /  //  %  **
```
Comparison Operators
```python
==  !=  >  <  >=  <=
```
Logical Operators
```python
and  or  not
```
Assignment Operators
```python
=  +=  -=  *=  /=
```
Bitwise Operators
```python
&  |  ^  ~  <<  >>
```
---
7️⃣ Binary, Octal & Hexadecimal Conversions
Python supports different number systems.
Number System	Base	Prefix
Binary	2	0b
Octal	8	0o
Hexadecimal	16	0x
Examples
```python
bin(10)
oct(10)
hex(10)

int('1010', 2)
int('12', 8)
int('A', 16)
```
